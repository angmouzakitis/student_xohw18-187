Steps to create NoC Firewall device
1)	make 			: to create /dev/sw_firewall module,
2)	make install 		: to insert module and provide access permission
3) 	cd ../TESTBENCHES	: to run any of the 5 testbenches (01_ to 05_)
			  	  a) cd to proper directory 
			  	  b) see the coresponding README.txt
4)	make clean 		: to remove module, device file, kernel logs
